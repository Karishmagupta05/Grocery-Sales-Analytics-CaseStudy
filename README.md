# Grocery-Sales-Analytics-CaseStudy
End-to-end data analytics case study on grocery retail sales data. The project focuses on solving real-world business problems such as outlet performance analysis, product category insights, location-based sales trends, and customer buying behavior using Excel.

## Project Overview
This project presents a complete data analytics case study on grocery retail sales data. The objective is to solve key business problems faced by a retail chain, such as identifying top-performing outlets, understanding product demand, evaluating location-based sales trends, and improving inventory planning. The analysis converts raw data into actionable business insights to support strategic decision-making.


## Business Objectives
Identify the best performing outlet types and sizes
Analyze product categories contributing highest revenue
Study impact of store location and visibility on sales
Understand customer preference for fat content products
Evaluate outlet age vs sales performance
Handle missing data issues for better analysis

## Dataset Information
The dataset contains 8,523 records with the following features:
Product Attributes
Item Type
Item Fat Content
Item Weight
Item Visibility
Outlet Attributes
Outlet Type
Outlet Size
Outlet Location Type
Outlet Establishment Year
Performance Metrics
Sales
Rating

## Key Business Problems Solved
Which outlet type generates maximum sales?
Does outlet size impact revenue?
Which product categories sell the most?
Does product visibility affect sales?
Which location tier performs best?
Low-fat vs regular product demand analysis
Outlet age vs performance study
Missing value treatment for Item Weight

## Tools & Techniques Used
Excel (Pivot Tables, Data Cleaning)
Data Visualization
Business Insight Reporting

## Key Insights
Supermarket Type1 outlets deliver the highest sales
Medium and large outlets outperform small stores
Fruits, snacks, and household products dominate revenue
Better shelf visibility improves sales
Tier 2 locations show strong market potential

## Business Recommendations
Expand high-performing outlet formats
Improve shelf placement for key products
Focus expansion in Tier 2 cities
Optimize inventory for top-selling categories
Strengthen data collection practices
